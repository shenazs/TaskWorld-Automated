TaskWorld Automated End-2-End TestCase
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

How to run test case ?
1. Clone or download project.
2. Re-install any missing references.
3. Rebuild the solution.
4. Open Test Explorer from Test>Windows.
5. Click on Run.

How to change test data ?
1. Navigate to Test Data folder.
2. Open 'CreateNewProject' excel file.
3. Enter valid entries to run test.
